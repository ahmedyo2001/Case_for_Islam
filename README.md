# Case for Islam

> An interactive, structured presentation of the rational and evidential case for Islam — built for non-Muslims who want to explore, and powered by the Muslim community.

---

## What is this?

**Case for Islam** is an open-source web application that walks non-Muslims through a logical, evidence-based argument for Islam.

It is not a collection of articles. It is not a chatbot. It is a structured, interactive argument tree — where each step builds on the last, and every claim is backed by curated, reviewed evidence.

The structure follows the **GORAP framework** — a da'wah methodology developed by iERA that guides a conversation from first principles to the foundations of Islam:

| Step | Topic |
|------|-------|
| **G** | God's Existence |
| **O** | God's Oneness |
| **R** | Revelation |
| **A** | Authenticity of the Quran |
| **P** | Prophethood of Muhammad ﷺ |

Each node in the tree contains multiple arguments and proofs, contributed by the Muslim community and moderated before going live.

---

## Who is it for?

**Non-Muslims** — anyone curious about Islam, skeptical, or genuinely searching. The experience is rational, respectful, and self-paced. No prior knowledge required.

**Muslims** — if you have a non-Muslim friend, family member, or colleague who wants to understand Islam logically, this is the link you send them.

---

## How it works

1. The user lands on the site and is presented with the first question: *Does God exist?*
2. They explore the arguments and proofs for that step
3. Once they are ready, they move to the next step
4. Each step only opens after engaging with the previous one
5. At any point they can go deeper into any argument or raise an objection

The goal is not to overwhelm — it is to guide, one step at a time.

## Contributing

This project has two sides:

### For Developers
We welcome contributions to the codebase. See [CONTRIBUTING.md](CONTRIBUTING.md) for setup instructions and open issues.

**Stack:**
- Frontend: React + Tailwind CSS
- Backend: Supabase (added in v1)
- Deployment: GitHub Pages

### For Muslims with Knowledge
If you have a strong argument, proof, or evidence to add to any section of the GORAP tree, you can submit it through the contribution form on the site.

All submissions are reviewed and moderated before going live. Only well-sourced, rational arguments are accepted — no emotional appeals, no weak evidence.

**What we are looking for:**
- Rational and philosophical arguments
- Scientific evidence where relevant
- Historical proofs
- Quranic and hadith references with proper sourcing

**What we are not looking for** (at this stage):
- Personal stories or anecdotes
- Emotional or spiritual appeals
- Unsourced claims

## Roadmap

**v0 — One day build**
Static site with the full GORAP tree and 2–3 arguments per section. No backend. Deployed and shareable.

**v1 — Contribution system**
Muslims can submit arguments via a form. Moderation pipeline. Dynamic content.

**v2 — Deeper experience**
Objection handling — common counterarguments addressed within each section. More arguments per node.

**v3 — Multilingual**
Arabic, French, Urdu and other languages.

## Motivation

There is no shortage of Islamic content online. But there is a shortage of content that is:

- Structured logically from first principles
- Interactive and self-paced
- Built collaboratively by the Muslim community
- Directed specifically at a non-Muslim audience seeking rational arguments

This project exists to fill that gap.
